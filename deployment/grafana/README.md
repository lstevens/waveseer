# WaveSeer Grafana Dashboard

This folder contains the Grafana dashboard configuration for monitoring the WaveSeer ingestion server.

## Dashboard Import

1. Open Grafana UI.
2. Go to **Create** → **Import**.
3. Upload `waveseer_dashboard.json` or paste its content.
4. Click **Load** and save to your preferred folder.

Adjust panel expressions, intervals, and layouts as needed.
